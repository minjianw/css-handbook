浏览器支持策略

手册在接下来将只更新IE6.0+, Firefox4.0+, Chrome4.0+, Safari4.0+, Opera15.0+（Opera从15.0开始转入webkit阵营）的支持数据，低于这些版本的数据将不再更新，让我们共同期望未来前端的生态环境越来越好。

版本更新

草案版本：v4.1.8(WD)

CSS参考手册目前正在v4.1.8草案中。广泛接受Issues和Pull Requests

最新版本：v4.1.7

更新时间：2015.03.09

  • 新增了 <image>, image(), image-set(), <gradient>, <color>, currentColor, inherit, initial, counters, font-size-adjust 参考；
  • 移除了 min(), max() 参考；
  • 修订了 <url>, linear-gradient(), repeating-linear-gradient(), radial-gradient(), repeating-radial-gradient() 参考；
  • 修订了 Dimension, Margin, Padding, Border, Background, Color, Font 等模块的浏览器兼容视图；
  • 修订了 Selectors, Rules, Values and Units, Appendix, CSS Hack 等模块的浏览器兼容视图；
  • 修订了 border-image, border-radius 及相关属性的参考描述；

上个版本：v4.1.6

更新时间：2015.01.28

  • 修订了 Positioning 模块的参考描述，使用新的浏览器兼容视图，增加部分移动浏览器支持；
